# Simulator

Simulação Python do hardware do Pulse — sem dependências de hardware físico.

## Passos

| Arquivo | O que faz |
|---|---|
| `passo1_bracelete.py` | Um bracelete imprimindo sensores no terminal |
| `passo2_*.py` | Em breve — múltiplos braceletes |
| `passo3_*.py` | Em breve — totens e fila de prioridade |
| `main.py` | Em breve — pipeline completo |

## Como rodar

```bash
python3 simulator/passo1_bracelete.py
```
